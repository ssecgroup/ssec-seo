🔍 ssec-seo API Documentation

The most advanced open-source SEO scanner - 100% FREE forever

GET /api/scan

Quick SEO scan (returns JSON)

curl "https://ssec-seo.vercel.app/api/scan?url=https://example.com"

Response:

{
  "status": "success",
  "url": "https://example.com",
  "pages_scanned": 1,
  "total_issues": 9,
  "critical_issues": 0,
  "high_issues": 1,
  "medium_issues": 2,
  "low_issues": 6,
  "score": 96,
  "risk_level": "low"
}

POST /api/scan

Full SEO scan (returns HTML report)

curl -X POST https://ssec-seo.vercel.app/api/scan \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com"}'

GET /api/scan?debug=1

Debug endpoint - check API status

curl "https://ssec-seo.vercel.app/api/scan?debug=1"

🔧 Integration Examples

JavaScript/Fetch

fetch('https://ssec-seo.vercel.app/api/scan?url=https://example.com')
  .then(r => r.json())
  .then(data => console.log(data.score));

Python

import requests
r = requests.get('https://ssec-seo.vercel.app/api/scan', 
                 params={'url': 'https://example.com'})
print(r.json()['score'])
